# Images Directory

This directory is for storing project-related images, diagrams, and visualizations.

## Purpose

You can push images here such as:
- Pipeline architecture diagrams
- DVC DAG visualizations
- Experiment comparison charts
- Model performance plots
- Project screenshots
- Documentation diagrams

## Usage

Simply add your image files to this directory and reference them in your documentation using:

```markdown
![Description](images/your-image.png)
```
